环境：

> python 3.7.1 64-bit
>
> Opencv 4.1.0
>
> Tesseract 5.0.0

> main.py	主文件用于图像处理
>
> typeimg.py	用于图像切割
>
> TypeORC_BaiduAPI.py	百度ORC
>
> TypeORC_Tesseract.py	Tesseract ORC
>
> decodeTest.py	unicode编码转汉字

> 摹释\_第一卷\_页数\_项\_列数\_行数.

> m_01_0002_a_01
> m_01_0002_b_01_01
> m_01_0002_c_01_01
